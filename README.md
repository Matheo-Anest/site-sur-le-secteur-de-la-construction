# 🏗️ Site Web : Le Secteur de la Construction

**Lien du site en direct :** [👉 Cliquez ici pour voir le projet](https://matheo-anest.github.io/site-sur-le-secteur-de-la-construction/)

## 🌐 Contexte du Projet
Ce site a été conçu comme une **SAE (Situation d'Apprentissage et d'Évaluation)**. L'objectif était de créer une plateforme d'information complète sur les métiers et les enjeux du secteur du bâtiment et des travaux publics.

## 🎨 Fonctionnalités
* **Navigation fluide :** Structure multi-pages (6 pages de contenu).
* **Responsive Design :** Adapté aux différents écrans grâce au CSS.
* **Sémantique HTML :** Code propre respectant les normes d'accessibilité.

## 🛠️ Stack Technique
* **Structure :** HTML5
* **Style :** CSS3 (Flexbox/Grid)
* **Hébergement :** GitHub Pages
